# رَصَد 🔴

لوحة مراقبة أحداث عالمية عربية — أخبار سياسية وعسكرية حية

## المميزات
- 📡 أخبار حية من BBC عربي، الجزيرة، العربية، سكاي نيوز، Reuters، NYT
- 🗺️ خريطة تفاعلية داكنة مع مؤشرات الأحداث
- 🔴 تصنيف تلقائي: نزاعات، عسكري، سياسي، دبلوماسي، أمني
- ⚠️ نظام خطورة: حرج، عالي، متوسط، منخفض
- 🔄 تحديث تلقائي كل 3 دقائق
- 📱 متوافق مع الأجهزة المحمولة

## التشغيل

npm install
